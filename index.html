¿Cómo obtener el token? (una sola vez)
1. Entra a github.com/settings/tokens
2. Generate new token → Fine-grained
3. En permisos, activa solo Gists: Read and write
4. Genera, copia el token y pégalo arriba
El token se guarda solo en este dispositivo, nunca en el código.
